Bir web sayfasının oluşturulmasında kullanılan HTML, .html ya da .htm uzantılarıyla ifade edilmektedir. Kodlama dilleri ile sıkça karıştırılan HTML, aslında betik bir dildir ve ‘tag’ yani etiket yapısından oluşturmaktadır. Açılan etiketlerin içerisinde yerleştirilen komutlar ile web sayfasının temel yapısını inşa etmekte kullanılan HTML, standart oluşturulmuş herhangi bir metnin web tarayıcıda görüntülenmesini sağlamaktadır. “Cascading Style Sheets” yani bilnen adıyla CSS, Türkçe’de “Basamaklı Biçim Sayfaları ya da Basamaklı Stil Şablonları” olarak adlandırılmaktadır. CSS, HTML tarafında oluşturulan ve web sayfasında gönderilen metinlerin renk, biçim, yazı fontu, arka plan, genişlik ya da yükseklik gibi parametreler ile şekillendirilmesini sağlamaktadır. “ID” ve “Class” adı verilen seçiciler ile verilen komutları uygulayan CSS, HTML etiketleri içerisinde kullanılmakta ve web sayfasının düzenlemesine dair tüm detayların oluşturulmasında destek sunmaktadır. Web Tasarımı ve Programa alanına ilgi duyanların ilk karşılaştıkları bu iki metod aracılığıyla bireysel ya da kurumsal tercihler HTML ve CSS solver araçlarıyla çözümlenerek web sayfasının ara yüzünde kullanıcılara ulaşmaktadır. Bu nedenle hem HTML etiketleri hem de CSS seçicileri bir web sayfasının iskeleti olarak adlandırılır. Aynı zamanda fotoğraf, video, gif vb. ara formlarında bir web sayfası üzerinden kullanıcılara sunulmasında rol oynayan HTML, kısa sürede dolu bir içerik oluşturmaya çalışan herkesin en büyük yardımcılarından birisidir. HTML, bütün bu işlevlerinin yanı sıra arama motorlarının web sitesi hakkında en fazla bilgiyi edindiği alandır. HTML etiketlerinin SEO, yani aramam motorunun optimizasyonuna uygunluğu, sayfanın kısa sürede indekslenerek kullanıcıların erişimine açılmasında da etkili olmaktadır. Bir bilgisayar içerisindeki Notepad, Visual Studio Code vb. standart yazım sayfaları ile oluşturulabilen HTML ve CSS komutları, web sayfalarına eklendiğinde işlevsel kazanmaktadır.
